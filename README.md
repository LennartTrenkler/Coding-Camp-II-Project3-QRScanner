# Coding-Camp-II-Project3-QRScanner

QR scanner MVP built with C++, Visual Studio, and OpenCV.

## Current features
- Select an image with a Windows file dialog
- Detect and decode one QR code
- Show the decoded content in the console

## Current structure
- `main.cpp` handles program flow
- `FileDialog.*` handles image selection
- `QRScanner.*` handles QR detection and decoding
